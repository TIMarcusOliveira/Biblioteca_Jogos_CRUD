Bem-vindo à GameLib

Organize e gerencie sua biblioteca de jogos facilmente!

Sobre o Sistema

Este sistema foi desenvolvido por Marcus Vinicius de Paula Oliveira como projeto de estudo para a disciplina de Sistemas Web ministrada pelo professor Robson Palmeirense no curso de Informática da Etec de Santa Isabel.

O objetivo do projeto é permitir o cadastro, consulta, atualização e exclusão (CRUD) de jogos em uma biblioteca virtual, utilizando PHP com PDO e banco de dados MySQL.

Funcionalidades

Contas Individuais

Cada usuário possui seus próprios jogos cadastrados.

Sessões e Cookies

O sistema usa sessões com cookies para manter o login ativo.

CRUD de Jogos

Cadastre, visualize, edite e remova seus jogos facilmente.
