Sobre o sistema

Este sistema foi desenvolvido por Marcus Vinicius de Paula Oliveira como projeto de estudo para a matéria de Sistemas Web, ministrada pelo professor Robson Palmeirense no curso de Informática da Etec de Santa isabel. O objetivo deste sistema é permitir o cadastro, consulta, atualização e exclusão (CRUD) de jogos em uma biblioteca virtual, utilizando PHP com PDO para interagir com um banco de dados MySQL.
